# Личный сайт

Одностраничник без сборки и зависимостей: `index.html` — это весь сайт.

Публикуется через GitHub Pages с ветки `main`.

**Как править:** блок `CONFIG` в начале `<script>` — контакты и данные.
Контент секций — массивы `SERVICES`, `FAQ`, `BIZ`, `RT`, `ARCH` там же.
Цвета, шрифты и ритм — `:root` в начале `<style>`.

**Как посмотреть локально:** открыть `index.html` двойным кликом.
